# Outreach Messages

Use these for builders working on Codex skills, MCP tools, agent workflows, or prompt/tool routing.

## GitHub Issue / Discussion Comment

```text
Hi, I am working on a local-first routing kit for Codex skills/plugins:
https://github.com/juew/Skill-Routing-Kit

It focuses on missed triggers, negative examples, and explainable routing rather than adding another runtime agent layer.

I noticed your work touches related agent/tooling problems. Curious if you have seen cases where the right capability was installed but did not trigger at the right time.
```

## Email

Subject:

```text
Question about skill/tool routing in agent workflows
```

Body:

```text
Hi,

I am working on Skill Routing Kit, a local-first Codex plugin for improving skill/plugin hit rate:
https://github.com/juew/Skill-Routing-Kit

The project focuses on a narrow problem: when many capabilities are installed, how should an agent decide which one should be primary, which should be helper context, and when a tempting tool should not be used?

I am collecting missed-routing examples from people building agent workflows, MCP tools, and Codex skills.

If you have seen this problem, I would appreciate any example or critique of the approach.

Thanks.
```

## What Not To Do

- Do not ask directly for a star.
- Do not paste the same comment into many unrelated repos.
- Do not pitch it as a general agent framework.
- Do not interrupt maintainers with long explanations.
- Do not claim production maturity beyond v0.1.0.
