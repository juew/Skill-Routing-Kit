# Launch Pack

This folder contains copy-ready launch material for Skill Routing Kit.

## Launch Goal

Reach people who already feel the pain of missed Codex skill/plugin triggers.

Do not pitch Skill Routing Kit as a universal agent platform. Pitch it as a small local-first routing kit that makes Codex pick the right skill or plugin more often.

## Recommended Sequence

1. GitHub internal setup: topics, release, discussions.
2. Record the 30-second demo GIF.
3. Publish the Chinese posts on V2EX and Juejin.
4. Publish the English posts on Hacker News and Reddit.
5. Post the X thread with the GIF.
6. Save Product Hunt for after the GIF and at least one discussion thread exist.

## Files

- `github-release.md`: GitHub release copy and repo setup checklist.
- `v2ex-post.md`: Chinese V2EX post.
- `juejin-post.md`: Chinese Juejin article.
- `hacker-news.md`: Show HN title and body.
- `reddit-posts.md`: Reddit post variants.
- `x-thread.md`: English X thread.
- `product-hunt.md`: Product Hunt launch copy.
- `gif-script.md`: GIF storyboard and recording commands.
- `outreach.md`: Cold outreach messages for related builders.
- `launch-checklist.md`: Step-by-step launch checklist.

## Positioning

One-liner:

> Skill Routing Kit makes Codex pick the right skill or plugin more often.

Short description:

> A local-first Codex plugin that improves skill/plugin hit rate with a routing guard, capability registry, and explainable diagnostics.

Chinese:

> 一个本地优先的 Codex 插件，用 routing guard、capability registry 和可解释诊断提高 skill/plugin 命中率。
