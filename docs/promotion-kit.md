## Short Description

Skill Routing Kit is a local-first Codex plugin that improves skill/plugin hit rate with a routing guard, capability registry, and explainable diagnostics.

## Longer Description

Codex can have many skills and plugins installed, but the right one does not always trigger. Skill Routing Kit adds a lightweight local routing layer: it classifies the request, recalls candidate capabilities from a registry, reranks by final artifact/source/action/risk, and explains why a skill should or should not be used.
